# Tenzies

A fun dice game built with React.

## Description

Roll the dice until all dice show the same number. Click on dice to hold them.

## Installation

Clone the repo and install dependencies:

```bash
npm install
```

## Running

Start the development server:

```bash
npm run dev
```

Open http://localhost:5173 in your browser.

## Build

To build for production:

```bash
npm run build
```
